Module config

This module provides access to the game's configuration information, as provided in config.txt.

The game configuration cannot be changed from Lily. This is intentional - changing the title/width/height would only be useful for games wanting to mess with the game window for effect, which is out of scope for Berry, and changing the framerate is just generally inadvisable.

Vars

var title: String

The title of the game, displayed in the window titlebar and in error messages.

var width: Integer

The width of the game, in pixels.

var height: Integer

The height of the game, in pixels.

var framerate: Integer

The framerate the game updates at.
