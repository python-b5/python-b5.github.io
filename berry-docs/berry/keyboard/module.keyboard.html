Module keyboard

This module contains routines for keyboard input.

See mouse and gamepad for similar modules that facilitate other kinds of input. All three modules have similar interfaces.

Enums

Key A key to be passed to an input function.

Functions

define held(key: Key): Boolean

Returns whether a given Key is held down.

define pressed(key: Key): Boolean

Returns whether a given Key was pressed this frame.

define released(key: Key): Boolean

Returns whether a given Key was released this frame.
