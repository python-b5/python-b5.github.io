Class Color

An RGB color.

For convenience, this class' constructor takes an integer as an argument, which is then broken down into the individual color channels. This means the standard hexadecimal color syntax can be used (eg. 0xffffff for white).

Errors
  • ValueError if color is not between 0 and 16777216 (or 0x000000 and 0xffffff, if using hexadecimal syntax).
class Color
{
    public var @red: Byte
    public var @blue: Byte
    public var @green: Byte
}
