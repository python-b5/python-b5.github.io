Module window

This module contains routines to modify the window.

Functions

define get_scale: Integer

Returns the current window scale factor.

define set_scale(scale: Integer)

Scales the window to a new factor.

scale must be between 1 and max_scale.

If the game is in fullscreen mode, this function will have no visible effect. However, the scale change will be apparent once fullscreen mode is next exited.

define get_fullscreen: Boolean

Returns whether the game is in fullscreen mode.

define set_fullscreen(fullscreen: Boolean)

Enters or exits fullscreen mode.

Vars

var max_scale: Integer

The largest window scale factor that can fit on the user's display.
