Module mouse

This module contains routines for mouse input.

See keyboard and gamepad for similar modules that facilitate other kinds of input. All three modules have similar interfaces.

Enums

Button A mouse button to be passed to an input function.

Functions

define held(button: Button): Boolean

Returns whether a given Button is held down.

define pressed(button: Button): Boolean

Returns whether a given Button was pressed this frame.

define released(button: Button): Boolean

Returns whether a given Button was released this frame.

define x: Integer

Returns the current X position of the mouse cursor, relative to the game window.

define y: Integer

Returns the current Y position of the mouse cursor, relative to the game window.
