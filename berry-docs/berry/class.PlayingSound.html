Class PlayingSound

An instance of a Sound being played.

This class cannot be constructed directly. PlayingSound objects are returned by Sound.play, and represent individual playback instances - there is no limit to the amount of PlayingSound objects that can be created from a single Sound.

Errors
  • For all methods except .finished: RuntimeError if self has finished playing.

constructor: PlayingSound

public define is_paused: Boolean

Returns whether self is paused.

public define pause

Pauses playback for self.

public define resume

Resumes playback for self.

public define is_looping: Boolean

Returns whether self is set to loop upon finishing.

public define set_looping(looping: Boolean)

Changes the looping state for self.

public define get_position: Double

Returns the current playback position for self in seconds.

public define set_position(position: Double)

Sets the playback position for self.

position is the desired playback position, in seconds.

Errors
  • ValueError if position is not between 0.0 and the length of the sound.

public define get_volume: Double

Returns the current volume of self, as a multiplier applied to the sound's original volume.

public define set_volume(volume: Double)

Sets the playback volume for self.

volume is the desired volume, as a multiplier applied to the sound's original volume.

Errors
  • ValueError if volume is less than 0.

public define get_speed: Double

Returns the current playback speed of self, as a multiplier applied to the sound's original speed.

public define set_speed(speed: Double)

Sets the playback speed for self.

speed is the desired speed, as a multiplier applied to the sound's original speed.

Errors
  • ValueError if speed is not between 0.0 and 100.0 (the latter limit is inherited from SDL, which Berry uses internally - if a different library were used for sound playback instead, that limit would likely not be in place).

public define stop

Immediately stops playback of self.

public define finished: Boolean

Returns whether self has finished playing.

Once this method returns true, there is no more use for self, and it can be safely discarded.

If self is looping, this method will never return true without intervention. You must stop playback manually with PlayingSound.stop.

All other methods will raise a RuntimeError if called after finishing, so check the result of this method first before calling them!
