Enum Color

An RGBA color.

Internally, colors are stored as four bytes - one for each channel. Often when working with colors, though, that is not the optimal format - bytes are clunky to work with, and in many cases you may not care about all four channels. This enum is provided to solve these issues.

Individual channels are always passed as Doubles in Berry's API, to make colors easier to work with. They are converted to bytes internally when being used for drawing (eg. 0.0 -> 0x00, 1.0 -> 0xFF).

To help make colors easier to construct, this enum provides a few variants depending on your use case:

  • RGB takes an Integer as an argument, which is then broken down into the individual color channels. This means the standard hexadecimal color syntax can be used (eg. 0xffffff for white). Colors using this variant are always opaque. Use this variant if you only want to set color, and don't care about transparency.
  • RGBA is like RGB, but also includes the alpha channel (as the least significant byte). This variant probably is only useful if converting back to a Color after calling .to_i.
  • Alpha takes a Double as an argument, which Berry will convert to a byte for drawing. Colors using this variant are always white. Use this variant if you only want to set transparency, and don't care about color.
  • Mixed is a combination of RGB and Alpha. It can be used to set both color and transparency at the same time in a more natural way than with RGBA or Channels.
  • Channels takes four Doubles as arguments, which are converted to bytes and used for each channel. This variant likely will not see much use, but it is provided as the inverse of the methods that return the individual channel values.
Errors

Errors may be raised when a method is called on self, when self is passed to Sprite.set_pixel or Sprite.draw, and when self is returned from a shader function.

  • For RGB and RGBA: ValueError if the color value is not between 0 and 16777216 (or 0x000000 and 0xffffff, if using hexadecimal syntax).
  • For Alpha and RGBA: ValueError if the alpha value is not between 0.0 and 1.0.
enum Color
{
    RGB(Integer),
    RGBA(Integer),
    Alpha(Double),
    Mixed(Integer, Double),
    Channels(Double, Double, Double, Double)
}

define red: Double

Returns the red channel of self as a Double.

define green: Double

Returns the green channel of self as a Double.

define blue: Double

Returns the blue channel of self as a Double.

define alpha: Double

Returns the alpha channel of self as a Double.

define color: Integer

Returns the red, green, and blue channels of self as an Integer, with each channel using one byte.

This is the same format that can be passed to the RGB variant.

define to_i: Integer

Converts self to an Integer, with each channel using one byte.

This is the same format that can be passed to the RGBA variant.
