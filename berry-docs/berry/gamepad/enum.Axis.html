Enum Axis

A gamepad axis to be passed to an input function.

An axis is anything on a gamepad that should be represented as a floating point number. This includes joysticks, ranging from -1.0 (left/top) to 1.0 (right/bottom), and triggers, ranging from 0.0 (released) to 1.0 (fully depressed).

enum Axis
{
    LeftStickX,
    LeftStickY,
    RightStickX,
    RightStickY,
    LeftTrigger,
    RightTrigger
}
