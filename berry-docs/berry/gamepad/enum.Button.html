Enum Button

A gamepad button to be passed to an input function.

Some gamepads have more buttons than this. This enum only contains the subset that are found on the majority of gamepads.

Bottom, Right, Left, and Top are the face buttons. They are named with their directions to make it unambiguous which is which, since different gamepads refer to these buttons by different names.

enum Button
{
    Bottom,
    Right,
    Left,
    Top,
    Select,
    Guide,
    Start,
    LeftStick,
    RightStick,
    LeftShoulder,
    RightShoulder,
    DPadUp,
    DPadDown,
    DPadLeft,
    DPadRight
}
