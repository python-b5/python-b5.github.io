Module gamepad

This module contains routines for gamepad input.

See keyboard and mouse for similar modules that facilitate other kinds of input. All three modules have similar interfaces.

Errors

These apply for all functions except .connected_gamepads.

  • ValueError if gamepad is less than 0.
  • ValueError if less than gamepad + 1 gamepads are connected.

Enums

Button A gamepad button to be passed to an input function.
Axis A gamepad axis to be passed to an input function.

Functions

define connected_gamepads: Integer

Returns the current number of connected gamepads.

This function should be used to verify which gamepad indices are available for use. Gamepad indices range from 0 to connected_gamepads() - 1.

define held(gamepad: Integer, button: Button): Boolean

Returns whether a given Button is held down on a given gamepad.

define pressed(gamepad: Integer, button: Button): Boolean

Returns whether a given Button was pressed this frame on a given gamepad.

define released(gamepad: Integer, button: Button): Boolean

Returns whether a given Button was released this frame on a given gamepad.

define get_axis(gamepad: Integer, axis: Axis): Double

Returns the value of a given Axis on a given gamepad.
